In HTML, a colour can be specified by using a colour name. HTML supports 140 standard colour names.

Link to full list of Colour Names supported by all modern browsers.

In addition to the predefined colour names, colours can be specified in other ways, the main one of which is HEX. This gives you access to infinite colours.

Note that we use the American COLOR rather than COLOUR when defining these.

Here are some examples:

Tomato background colour on a paragraph.

Heading 1 in Orange or you could use the HEX code #FFA500

Heading 2 in DodgerBlue or HEX code #1E90FF

Heading 3 in MediumSeaGreen or HEX code #3CB371

Heading 4 in Gray or HEX code #808080 (Note Gray OR Grey works)

Heading 5 in SlateBlue or HEX code #6A5ACD
Heading 6 in Violet or HEX code #EE82EE

In HTML, a comment is a section of text that is not processed by the web browser. Comments are enclosed in tags and are a very useful way or organising your coding.
