List of some basic HTML tags:

  • The<!DOCTYPE html> declaration defines that this document is an HTML5 document
  • The<html> tag represents the root of an HTML document
  • The<head> tag contains meta information about the HTML page
  • The<title> tag specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
  • The<style> tag is used to define style information (CSS) for a document. This is where you specify how HTML elements should render in a browser
  • The<body> tag defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, lists, etc.
  • The<h1>to<h6> tags define headings
  • The<p> tag defines a paragraph
  • The<b> tag specifies bold text without any extra importance
  • The<strong> tag is used to define text with strong importance. It will appear the same as bold
  • The<i> tag makes text italic
  • The<br> tag represents a line break. It is an empty tag, which means that it doesn't have a closing tag
  • The<ol> tag represents an ordered list. The list items will be marked with numbers by default
  • The<ul> tag represents an unordered list. The list items will be marked with bullets by default
  • The<img> tag is used to embed an image in a web page. It is an empty tag, which means that it doesn't have a closing tag.
    It has two required attributes:
    • src - Specifies the path to the image
    • alt - Specifies an alternate text for the image
  • The<div> tag defines a division or a section in an HTML document and is used as a container for HTML elements
  • The<span> tag is used when you want to style a particular part of your content differently
  • The<a> tag defines a hyperlink.
    The most important attribute of the <a> element is the href attribute, which indicates the link's destination.
    The link text is the part that will be visible to the reader. Clicking on the link text, will send the reader to the specified URL address.
