  • I included a DOCTYPE - They haven't been mentioned yet
  • I added lang="en" to the html tag - That hasn't been mentioned yet
  • I put the style in the head, rather than in the body as per Bernardine's instructions. This is my preference.
  • I have used CSS to make all the required changes in order to keep the HTML clean and uncluttered
  • I have included comments in the CSS explaining what each does
  • In this exercise, there is only ONE occurance of H1, H2, H3, UL and HR
  • I mention this because it has an impact on how I have written my CSS
  • The way the CSS is written for these tags means that EVERY occurance of these will be styled in that fashion
  • This might not be desirable and may have to be changed as the exercise continues
  • There are, however, TWO occurances of DIVs
  • This also had an impact on how I wrote my CSS
  • For these I made classes as I didn't want them styled in the same way
  • One I named 'friday' and the other I named 'center'
  • In the html, I applied the relevant class to the relevant div
  • You may wonder why I put the pizza image in a div
  • It was neccesary to hold the image in the centre of the page AND ensure that ONLY the image was clickable when linked
  • Without doing this, the entire width of the page, either side of the image would be clickable
